◆ DSCRS_PER_CHN

#define DSCRS_PER_CHN   (3u)

Number of DMA descriptors required per DataWire channel:

  1. First one is a 2-D descriptor which is used to move full data packets.
  2. Second one is a 1-D descriptor which moves the DWORD aligned portion of any short packet which completes the transfer.
  3. Third one is a 1-D descriptor used to transfer the remaining non-DWORD aligned data (1, 2 or 3 bytes) where required.
